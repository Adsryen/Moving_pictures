# Moving_pictures
**一个会动的图片**
<br>
<br>
支持上传自定义图片
支持调节眼睛及其嘴巴的位置
<br>
环境要求：
能正常显示的服务器或者虚拟主机即可


==================================
<br>
Support for uploading custom pictures. 
Support to adjust the position of eyes and mouth. 
<br>

Environmental requirements: 
A server or virtual host that can be displayed normally
